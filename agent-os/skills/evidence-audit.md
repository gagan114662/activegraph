# Skill: Evidence Audit

Use before any status, review, or completion claim.

Checklist:
1. List every predicate.
2. Map each predicate to file/hash/command output.
3. Run the required commands.
4. Paste literal output into the relevant log.
5. Mark unknown/unrun as incomplete, not green.
6. Confirm git status and pushed head.

Output:
- predicate-to-evidence table
- literal commands
- unresolved gaps
