# Skill: Evidence Indexing

Use when evidence is split across outer workspace, inner repo, chat, Pentagon
logs, or untracked files.

Steps:
1. Name the authoritative repo for each claim.
2. Name the authoritative artifact path for each frame.
3. Record related commits and pushed heads.
4. Mark stale, duplicate, untracked, or chat-only artifacts as non-proof.
5. Classify every frame as closed, blocked, superseded, retired, or
   pre_satisfied_without_frame.
6. Update frames/evidence-index-2026-05-22.md or the current evidence index.

Output:
- authoritative artifact table
- stale/untracked proof list
- autonomy verdict separate from product correctness
